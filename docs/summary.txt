Enterprise AI Agent Platform

项目简介

Enterprise AI Agent Platform（企业级 AI Agent 平台）是一个面向企业内部研发团队的智能 AI 平台，为企业提供统一的 AI 服务入口。

项目采用前后端分离架构，基于 LangChain、LangGraph、RAG、 Function Calling等技术，实现企业知识库问答、数据库智能查询、联网搜索、工具调用等能力，并支持 Docker Compose 一键部署。

平台采用 Agent Router + Workflow 的架构设计，根据用户意图自动选择不同的 Agent 工作流，具有良好的可扩展性，能够方便地扩展新的 Agent、Tool 和业务流程。

---

核心功能模块

1. Agent Router

- 用户意图识别
- 自动路由至对应 Workflow
- 支持 RAG、SQL Agent、Web Search、Tool Calling 等不同业务流程

2. 企业知识库（RAG）

- 支持 PDF、Word、Markdown、TXT 等文档上传
- 文档解析、Chunk 切分、Embedding、向量检索
- 基于 LangChain 构建 Retrieval Pipeline
- 支持引用来源，提高回答可信度

3. LangGraph Workflow

- 使用 LangGraph 编排 Agent 工作流
- 将 RAG、Tool Calling、SQL Agent 等流程解耦
- 支持后续快速扩展新的 Agent 与 Workflow

4. Tool Calling

- 支持 MCP 或 Function Calling
- 可调用 HTTP API、文件系统等外部工具
- 实现 AI 与外部系统协同

5. SQL Agent

- 基于自然语言查询 PostgreSQL
- 自动分析数据库 Schema
- 自动生成 SQL 并执行查询
- 支持 SQL 安全限制，仅允许只读操作

6. Web Search

- 支持联网搜索最新公开信息
- 与企业私有知识库结合，提高回答完整性

7. Conversation Memory

- 保留当前会话上下文
- 提高连续对话体验
- 不设计复杂长期记忆，降低系统复杂度

8. 权限管理（RBAC）

- JWT 用户认证
- 普通用户与管理员角色管理
- 支持知识库管理、模型配置、API Key 管理等后台功能

项目亮点

- 基于 Agent Router 实现智能任务路由，提高系统扩展性
- 使用 LangGraph 构建模块化 Agent Workflow
- 基于 LangChain 实现 RAG、Prompt、Retriever、Tool 等核心组件
- 集成 SQL Agent、Web Search 与 Tool Calling，实现多场景 AI 能力
- 支持 MCP/Function Calling，实现 AI 与外部工具协同
- 前后端分离架构，支持 Docker Compose 一键部署，具备企业级应用开发能力

Enterprise AI Agent Platform 开发路线

版本| 核心目标| 实现功能
v0.1| Chat + SSE + FastAPI + React| 完成项目基础框架搭建，实现前后端分离；支持用户与 LLM 对话；使用 SSE 实现流式输出，具备类似 ChatGPT 的实时回答体验。
v0.2| LangChain + LangGraph + Agent Router + Function Calling（简单 Tool）| 引入 LangChain 与 LangGraph 重构聊天流程；使用 PromptTemplate、ChatModel 等 LangChain 组件；搭建 Agent Router，实现请求路由；支持 Function Calling，并接入简单工具（如获取当前时间、计算器），验证 Agent 调用 Tool 的完整流程。
v0.3| RAG（Retriever Tool）| 集成企业知识库；支持 PDF、Word、Markdown、TXT 等文档上传；完成文档解析、Chunk 切分、Embedding、向量存储与检索；将 Retriever 封装为 Tool，由 Agent 自动调用，实现企业知识问答，并支持引用来源。
v0.4| SQL Agent + Conversation Memory| 接入 PostgreSQL；支持自然语言生成 SQL 并查询数据库；增加 SQL 安全校验（禁止危险操作）；引入 Conversation Memory，实现当前会话上下文记忆，支持连续追问。
v0.5| MCP（HTTP Tool、File Tool 等）| 引入 MCP；接入 HTTP API、文件系统等 MCP Server；统一 Tool 调用方式，使平台具备标准化工具扩展能力，并支持后续快速增加新的外部工具。
v0.6| Web Search Tool| 集成联网搜索工具；支持获取互联网实时信息；结合企业知识库与公开信息，实现更加完整、准确的回答。
v0.7| JWT + RBAC| 增加用户登录认证；实现 JWT 身份验证；完成 RBAC 权限控制，区分普通用户与管理员；管理员可管理知识库、模型配置等资源。
v0.8| 后台管理（知识库、模型、用户）| 完成企业后台管理模块；支持用户管理、知识库管理、聊天记录管理、模型切换、API Key 配置等功能，形成完整的平台管理能力。
v1.0| Docker Compose + 完整部署 + README + API 文档| 完成项目部署方案；支持 Docker Compose 一键启动；完善 README、API 文档和项目说明；整理项目目录结构，达到可演示、可部署、可交付的正式版本。
